# วิธีติดตั้ง

### Windows

### Video สอนลง https://youtu.be/UZptdL5jHEM

1. ไปโหลด Node ที่ https://nodejs.org/en/download/
2. ติดตั้งให้เรียบร้อย
3. เปิด INSTALL.bat
4. เรื่มโดนการเปิดไฟล์ `START.bat`

### Linux/MacOS

1. install node
2. npm install
3. npm start

### Contribute

1. มี yarn | npm i -g yarn

```sh
yarn
```

ลอง

```sh
yarn try
```
